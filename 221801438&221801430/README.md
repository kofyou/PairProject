# 结对第二次作业——顶会热词统计的实现

* [作业博客链接](https://www.cnblogs.com/chutiiing/p/14604211.html)
* 结对学号：221801438 & 221801430

## 项目介绍
* paper search
 * 功能1：对已爬取的论文列表进行操作
    - 可对论文列表进行删除；
    - 可对论文列表进行查询详细信息（支持模糊查询，查询结果的展示、排序功能）;

* 功能2：分析已爬取到的论文信息，提取top10个热门领域或热门研究方向
    - 形成如关键词图谱之类直观的查看方式；
    - 可对多年间、不同顶会的热词呈现热度走势对比，以动图的形式呈现;
    
* 2. 附加功能
    - 支持用户输入单个论文题目进性查询；
    - 实现登录功能，一个用户对应一个用户文章列表
